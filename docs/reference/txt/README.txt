This documentation is organized into separate files, each corresponding to a
specific Python file within the src/django_auth0_engine directory. This
structure provides a clear and modular organization, making it easier for users
to navigate and find relevant information.

apps.txt
	Django app configuration.

auth_engine.txt
	Details of AuthEngine and AuthVerifierEngine.

cfg.txt
	A module to share information among other modules.

exceptions.txt
	Details of  AuthEngineError.

management_engine.txt
	Details of ManagementEngine.

middleware.txt
	Details of SessionAuthMiddleware and HeaderAuthMiddleware.

oidc.txt
	Details of OIDCClaimsStruct.

response.txt
	Details of AuthEngineResponse.

user.txt
	Details of User.
